# vue-to-do-app
Simple ToDo List created using Vue.js :)

## How to use
1. Clone this repository and move to the directory.
```
$ git clone https://github.com/kazumi3858/vue-to-do-app.git
```

2. Run the following command to open the html file.
Linux
```
$ xdg-open index.html
```

Mac
```
$ open index.html
```
